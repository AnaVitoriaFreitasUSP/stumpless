Detailed Description

Socket targets allow logs to be sent to a Unix domain socket, like the frequently used /dev/log or /var/run/syslog sockets associated with syslog daemons.

Opens a socket target.

Parameters
nameThe name of the socket to send logs to.
local_socketThe name of the Unix domain socket to connect from. This can be set to NULL if you'd like to simply let the target generate one.
optionsThe options to use for this target. This should be a bit-wise or of STUMPLESS_OPTION values. Use STUMPLESS_OPTION_NONE if you do not need any options.
default_facilityThe default facility to use on messages logged to this target without a facility code of their own. This should be a STUMPLESS_FACILITY value.
