Detailed Description

Windows Event Log (WEL) targets allow messages to be sent to the Windows event handling system.

Opens a WelTarget on the local machine.

Parameters

optionsThe options to use for this target. This should be a bit-wise or of STUMPLESS_OPTION values. Use STUMPLESS_OPTION_NONE if you do not need any options.

Opens a WelTarget on a remote machine.

Parameters
serverThe remote server to log to. This should use the Universal Naming Convention (UNC) of the remote end. If this parameter is set to NULL, then the local machine is used. This is equivalent to using the two argument constructor.
nameThe name of the target to open. This should match the name of the Windows Event Source that the logs will be added to.
optionsThe options to use for this target. This should be a bit-wise or of STUMPLESS_OPTION values. Use STUMPLESS_OPTION_NONE if you do not need any options.
